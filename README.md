# StrainNode2D

**StrainNode2D** — это 2D-симулятор физики мягких тел и механизмов, построенный на архитектуре узлов и балок (Node-Beam architecture). Движок позволяет создавать сложные конструкции, мосты, автомобили и механизмы, а затем наблюдать за их реалистичной деформацией и разрушением в реальном времени.

Проект вдохновлен такими симуляторами, как Rigs of Rods и BeamNG.drive, перенося концепцию расчетов масс-пружинных сетей в двумерное пространство.

## 🚀 Особенности и технологии (Версия 0.1.0)

- **Реалистичная физика:** Симуляция масс, инерции, гравитации и аэродинамики (включая эффект Магнуса).
- **Масс-пружинная модель (Node-Beam):**
  - Обычные балки (с пределом текучести и прочности на разрыв).
  - Тросы (работают только на растяжение).
  - Гидравлические цилиндры (активное изменение длины).
- **Моторы и колеса:** Узлы с возможностью приложения крутящего момента для создания транспортных средств.
- **Продвинутое трение:** Честная модель сухого трения Кулона (разделение на статическое трение покоя и кинетическое трение скольжения).
- **Оптимизация столкновений:** Использование Spatial Hashing (пространственного хеширования) для Broad-phase поиска коллизий, что позволяет симулировать сотни узлов без падения FPS.
- **Продвинутый редактор:** Структурное копирование механизмов (с сохранением геометрии), рамка выделения, динамическая камера.
- **Технологический стек:** - Язык: `Python 3`
  - Рендер и окно: `Pygame`
  - Диалоговые окна: `Tkinter`

# 🎮 Управление

В симуляторе реализован функциональный встроенный редактор для сборки на лету.

### Камера и Время
* **Скролл мыши:** Приблизить / Отдалить (Zoom).
* **Зажатая ЛКМ (по пустому месту):** Перемещение камеры (Pan).
* **F:** Привязать камеру к выделенному узлу (следящая камера).
* **Пробел (Space):** Пауза симуляции.
* **Стрелки Вверх / Вниз:** Ускорение и замедление времени (Slow-mo от 0.1x до 1.0x).

### Выделение и Строительство
* **ЛКМ по узлу:** Выделить / Перетащить объект (сохраняет инерцию при броске).
* **ЛКМ + Shift:** Множественное выделение.
* **ЛКМ (по пустому месту) + Shift:** Создать новый случайный узел.
* **Зажатая ПКМ:** Выделение рамкой (Box Selection).
* **ЛКМ + Ctrl (между узлами):** Быстрое создание балок между узлами (цепочное строительство).
* **1, 2, 3:** Изменить тип выделенной балки (Пружина, Трос, Гидравлика).
* **СКМ (Колесико мыши):** Заморозить / Разморозить выделенные узлы в пространстве (сделать статичными).

### Редактирование сцены
* **Ctrl + C / Ctrl + V:** Структурное копирование (копирует выделенные узлы и связи между ними, сохраняя форму механизма).
* **E:** Открыть меню Инспектора для изменения свойств выделенного узла или балки (масса, жесткость, радиус и т.д.).
* **Delete:** Удалить выделенные объекты (корректно разрывает связи).
* **F5 / F9:** Сохранить сцену / Загрузить сцену в формате `.json`.

### Управление механизмами
* **A / D:** Управление моторами (вперед/назад).
* **W / S:** Управление гидравлическими поршнями (выдвинуть/втянуть).

# 🛠 Запуск проекта

Убедитесь, что у вас установлен Python 3 и библиотека `pygame`.

1. Установите зависимости:
   ```bash
   pip install pygame
   ```
2. Запустите симулятор из корня проекта:
   ```bash
   python run.py
   ```
